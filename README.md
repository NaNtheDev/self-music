# Self Music

## Requirements
### ffmepeg (in your path)
Link: https://ffmpeg.org/download.html
### Nodejs
Link: https://nodejs.org/en/download/

## How to use ?
```sh
$ git clone https://github.com/NaNtheDev/self-music
$ npm i
$ node .
```
